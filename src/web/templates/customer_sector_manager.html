About This Tool

What is this tool?

Assigns sectors to customers so revenue can be grouped and analyzed by industry. Every customer should have a primary sector.

Sectors

Six standard groups: Commercial, Financial, Healthcare, Outreach, Political, and Other. Use "Add Sector" to create a new one if none of the existing options fit.

Assigning a sector

Click the sector dropdown on any customer row and select a sector. The assignment saves immediately. Use the "Unassigned Only" filter to find customers that still need a sector.

Bulk assignment

Check multiple customers using the checkboxes, choose a sector from the "Assign to..." dropdown, and click Apply. All selected customers are updated at once.

Revenue filter

Prioritize high-value customers by filtering to $100K+ first. Zero-revenue customers are often orphan records that may not need sector assignment.
